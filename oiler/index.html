Project Oiler

Similar to Project Euler, but focused on a wider range of simpler problems you're more likely to encounter in a work environment.

Useful if you want to:

  • Practice the skill of writing quick scripts to solve a broad range of problems
  • Try out a new programming language/ecosystem to see how well it works for general purpose code
